Project 3 SOHO Network Design

Overview

You will work in groups on this project.
The purpose of this project is to design and build a SOHO (small office/ home office) network.
You will do this by investigating the latest trends in network design on the Internet, and then using VMware (Workstation or Fusion-see Project 2) to create a series of virtual machines to mock up your network.

Assumptions:

  • Five full time employees
  • Hi speed wired internet access available
  • Security is a must, for both wireless and wired network
  • Firewall required, IPCop OK

Requirements:

  • Firewall - such as IPCop
  • Remote access for employees when traveling is required. Remote Desktop or VNC is acceptable.
  • Need web (http) and ftp servers.
  • Network Storage required (FreeNAS ok)
  • VOIP- Asterisk or equivalent
  • email- can be outsourced or locally hosted, you decide
  • Intrusion detection/Intrusion Prevention/Network monitoring -WireShark
  • Incorporate Smartphones into your Project (see references below)

Smartphone References:

  • Smartphones - Wikipedia
  • Business models -ZDNet
  • How Smartphones work - howstuffworks
  • Intel @ CES
  • Google App Inventor

Steps

  1. Review requirements, decide if there are any other you want to add.
  2. Develop network schematics (Use Visio)
    1. Logical
    2. Physical
  3. Decide on software required to meet requirements
  4. Decide on hardware required to host software
